# Módulos y campos (versión inicial)

Este documento lista los módulos de datos del alumno y los campos asociados. Es una primera versión sujeta a cambios y ajustes.

## 1. Datos Personales del Alumno

- Nombre completo
- Fecha y lugar de nacimiento
- Sexo
- Nacionalidad
- Dirección
- Datos de inscripción

Notas:

- “Datos de inscripción” se detallará en la siguiente iteración (ej.: grado/sección, fecha de ingreso, año escolar, etc.).

## 2. Datos médicos del alumno

- Problemas de audición y lenguaje
- Enfermedades
- Alergias
- Grupo sanguíneo
- Alimentación
- Medicamentos autorizados

## 3. Datos del Padre

- Nombre completo
- ¿Vive con el alumno?
- Fecha y lugar de nacimiento
- Cédula
- Estado civil
- Información laboral
- Número de contacto

## 4. Datos de la Madre

- Nombre completo
- ¿Vive con el alumno?
- Fecha y lugar de nacimiento
- Cédula
- Estado civil
- Información laboral
- Número de contacto

## 5. Datos del Embarazo

- Complicaciones durante el embarazo
- Tipo de parto
- ¿Fumó o bebió estando embarazada?
- Peso del niño al nacer
- Demás información

## 6. Datos del Representante Autorizado

- Nombre completo
- Edad
- Cédula
- Relación o parentesco con el alumno
- Número de contacto
- Demás personas autorizadas
